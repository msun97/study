컴퓨터 코드 요소

kdd 태그

kdd 태그는 "key developments and developments in data mining and knowledge discovery"를 의미한다. 이는 키보드 입력을 정의하는데 사용한다.

Ctrl + S

samp 태그

samp 태그는 "sample output from a computer program"를 의미한다. 이는 컴퓨터 프로그램의 출력을 정의하는데 사용한다.

File not found.
Press F1 to continue

code 태그

code 태그는 "computer code"를 의미한다. 프로그래밍 언어의 코드를 정의하는데 사용한다.

    
      function multiply(a, b) {
        return a * b;
      }

var 태그
