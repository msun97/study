id 속성 사용

id는 요소에 대해 고유한 이름을 지정합니다. 구문은 다음과 같습니다. #id_name{}

클래스와 ID의 차이점

클래스: HTML 요소의 공통 스타일을 지정합니다. id: HTML 요소의 고유 스타일을 지정합니다.

id와 북마크

id를 이용하면 HTML의 #요소id를 링크로 사용하여 해당 요소로 이동할 수 있습니다.
