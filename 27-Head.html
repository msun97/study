Head Element

Head 요소는 HTML 문서의 head section(헤더)에 위치합니다. Head 메타데이터는 HTML 문서에 대한 데이터다. 문서 제목, 문자 집합, 스타일, 스크립트 및 기타 메타 정보를 정의한다.

title

title 요소는 HTML 문서의 제목 표시줄, 페이지 탭에 표시되는 제목을 정의합니다. 이는 SEO에 매우 중요하다.

style

style 요소의 CSS는 HTML 요소에 대한 스타일을 정의합니다.

link

link 요소를 사용하면 CSS, JS, IMG, FONT, ETC. 파일을 HTML 문서에 로드할 수 있습니다.

meta

meta 요소는 HTML 문서에 대한 메타데이터를 정의합니다. meta 요소는 SEO, viewport, description, author, keywords, etc.를 정의할 수 있습니다.

뷰포트

viewport meta tag를 사용하면 HTML 문서를 viewport size(화면 크기)에 맞게 조절할 수 있습니다. viewport meta tag는 viewport-width, viewport-height, initial-scale, minimum-scale, maximum-scale, user-scalable, etc.를 정의할 수 있습니다.

script

script 요소를 사용하면 JavaScript를 HTML 문서에 로드할 수 있습니다.

base

base 요소를 사용하면 HTML 문서에서 링크를 relative path로 지정할 수 있습니다.
