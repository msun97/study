URL

URL(Uniform Resource Locator) : 인터넷상의 자원을 찾아내기 위한 경로

웹 브라우저는 URL을 사용하여 웹 서버에 페이지를 요청한다. 이는 다음과 같은 구문 규칙을 따른다.

scheme://prefix.domain:port/path/filname

scheme : http, https, ftp, file, mailto, etc.

prefix : http, ftp, mailto, etc.

domain : www.example.com

port : 80 (http), 443 (https)

path : /index.html

filname : index.html
