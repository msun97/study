Ordered List (ol)

ol 태그는 순서있는 목록을 정의합니다. li로 각 항목을 정의합니다.

  1. item 1
  2. item 2

Description List (dl)

dl, dt, dd로 description list를 정의합니다. dl는 설명 목록을 정의하고 dt는 용어(이름)을 정의하며 dd는 각 용어를 설명합니다.
