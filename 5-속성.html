HTML 속성

HTML 속성은 HTML 요소(element)의 고유한 속성을 부여합니다.

href 속성

a(anchor) 요소의 href 속성은 링크(hyperlink)의 URL을 설정합니다.

Google

src 속성

img(image) 요소의 src 속성은 이미지의 URL을 설정합니다.

너비 및 높이 속성

img(image) 요소의 width, height 속성은 이미지의 너비, 높이를 설정합니다.

alt 속성

img(image) 요소의 alt 속성은 이미지가 캡션(caption)으로 사용될 때, 이미지가 존재하지 않을 때, 또는 스크린 리더(screen reader)를 사용하는 사용자에게 보여지는 텍스트를 설정합니다.

스타일 속성

p(paragraph) 요소의 style 속성은 CSS(Cascading Style Sheets)로 CSS 스타일을 적용할 수 있습니다.

This is a paragraph with red color and bold font weight.

lang 속성

웹 페이지 언어를 선언합니다. 검색 엔진과 브라우저를 지원하기 위한 속성입니다.

제목 속성

title은 요소에 대한 제목을 추가로 정의합니다.
