반응형

Responsive Web Design (RWD)

RWD는 웹 페이지를 다양한 디바이스(PC, tablet, smartphone)에서 동일하게 보이도록 하는 것을 말합니다.

뷰포트 설정

meta 태그에 다음 태그를 추가하면 반응형 웹사이트를 만들 수 있다.

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

반응형 이미지

크기 속성을 %로 설정하면 이미지가 반응할 수 있다. 이는 이미지 이외의 모든 요소에도 적용된다.

max-width

max-width: 100%; CSS 속성을 사용하면 이미지가 100% 너비로 조절되며, 너비가 100% 너비보다 줄어들면 이미지가 축소된다.

picture 요소

picture 요소를 사용하면 srcset, sizes, type, alt, srcset, srcset의 세트(set)을 지정할 수 있다.

반응형 텍스트 크기

뷰포트 너비의 vw로 텍스트 크기를 설정한다. 1vw=1%의 뷰포트 너비다

미디어 쿼리

미디어 쿼리(media query)를 사용하면 CSS를 @media rule로 조건에 맞게 적용할 수 있다.

Framework

Bootstrap, Foundation, Pure.css, Semantic UI, Tailwind CSS, MDBootstrap, Bulma, Ant Design, Element UI, Vuetify, Reactstrap, Styled Components, Rimatch, and many more...
