CSS

CSS란?

CSS(Cascading Style Sheets)는 HTML과 XML 문서의 presentation(디자인)을 구성하기 위한 언어이다. CSS를 이용하면 HTML의 구조와 디스플레이, 스타일 등을 제어할 수 있다. 이때 cascading이라는 단어는 부모 요소에 적용 된 스타일이 자식 요소에도 적용된다는 것을 의미한다.

CSS 사용하기

인라인 style - html 요소 내부의 속성 이용

내부 - 섹션 태그 style을 사용하여 head에 작성

외부 - style.css 파일을 만들어 style.css 파일 내에 style을 작성한 후 link 태그를 head에 작성
