HTML 기본 예제

!DOCTYPE html : 이는 문서가 html5 형태로 작성되었음을 나타낸다

HTML 제목

h태그로 html 문서의 제목을 정의할 수 있다. 1~6까지 있으며, 숫자가 클수록 중요도가 떨어진다

HTML 문단

p태그로 html 문서의 문단을 정의할 수 있다. HTML5부터는 p태그를 사용하지 않아도 block 요소로 동작하므로 p태그를 정의 할 필요는 없다.

HTML 링크

a태그로 html 문서 내의 링크를 정의할 수 있다. href 속성으로 링크 URL을, target 속성으로 링크를 여는 위치(새 창, 현재 창)을 지정할 수 있다.

HTML 이미지

img태그로 html 문서 내의 이미지를 정의할 수 있다. src 속성으로 이미지 URL을, alt 속성으로 alt 텍스트를 지정할 수 있다.

HTML 소스 검사

크롬 브라우저에서 F12를 누르면 개발자 모드로 HTML 코드를 확인할 수 있다.
