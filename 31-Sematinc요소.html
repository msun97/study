Semantic Elements

정의

Semantic Elements는 HTML5에서 새롭게 추가된 요소로, 웹 페이지의 구조, 의미, 스타일링, 검색엔진 최적화(SEO)를 위해 HTML5의 기능을 확장합니다.

Examples

의미적 요소 예시

  • article
  • header
  • aside
  • footer
  • nav
  • section
  • figure
  • figcaption
  • time
  • mark
  • details
  • summary
  • main

section

section 요소는 HTML5에서 새롭게 추가된 semantic element로, content의 독립적인 콘텐츠 영역을 정의합니다. 일반적으로 제목이 있는 주제별 콘텐츠 그룹입니다. 챕터, 소개, 뉴스 항목, 연락처 정보 등에 사용할 수 있습니다.

article

article 요소는 HTML5에서 새롭게 추가된 semantic element로, content의 독립적인 자체 포함 콘텐츠 영역을 정의합니다. article 요소는 blog, news feed, forum, magazine, social media post, user comments, reviews, forums, etc. 에서 사용할 수 있습니다.

header

header 요소는 소개 내용, 탐색 링크 세트를 담는 컨테이너를 나타냅니다. 제목 요소, 로고, 저자 정보 등이 들어갑니다.

footer

footer 요소는 content의 맨 아래에 위치하며, site footer, copyright, navigation links, social media icons, etc. 에서 사용할 수 있습니다.

nav

nav 요소는 일련의 탐색 링크의 집합을 정의합니다.

aside

aside 요소는 content의 side bar, sidebar, related content, navigation sidebar, etc. 에서 사용할 수 있습니다.

figure 및 figcaption

figure는 그림, 다이어그램, 사진, 코드 목록 등과 같은 독립적인 콘텐츠를 지정합니다. figcaption은 이러한 figure의 캡션을 정의합니다.
