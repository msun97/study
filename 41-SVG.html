SVG

Scalable Vector Graphics

SVG(Scalable Vector Graphics)는 2D vector graphics technology로, XML(eXtensible Markup Language) 기반으로 작성할 수 있습니다. 이는 웹용 벡터 기반 그래픽을 정의하는데 사용됩니다. 확대/축소하거나 크기를 조정해도 품질이 손상되지 않습니다.

SVG vs Canvas

Canvas는 2D rendering context를 HTML5 canvas element 안에서만 사용합니다. SVG는 2D vector graphics technology로, SVG DOM 내에서 사용 가능합니다.
