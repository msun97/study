HTML Layout

레이아웃 요소

HTML5에서 HTML5 Layout 요소로 <header>, <nav>, <main>, <aside>, <footer>를 정의합니다.

  • <header>: 사이트의 헤더(header)를 정의합니다.
  • <nav>: 네비게이션(navigation) 링크를 정의합니다.
  • <main>: 사이트의 main content(본문)를 정의합니다.
  • <aside>: sidebar(비주요) content(부분)를 정의합니다.
  • <footer>: 사이트의 footer(footer)를 정의합니다.
  • <article>: 독립된 content를 정의합니다.
  • <section>: content section(섹션)을 정의합니다.
  • <details>: 사용자가 필요에 따라 열고 닫을 수 있는 추가 세부 정보를 정의합니다.

레이아웃 툴

  • CSS Framework
  • Grid System
  • float
  • Flexbox

Framework

W3.CSS, Bootstrp과 같은 CSS 프레임워크를 사용한다.

Grid System

CSS 그리드 레이아웃 모듈로 행과 열을 통해 레이아웃을 구성한다.

float

float 속성을 사용하여 block 요소를 side by side로 배치한다.

Flexbox

flexbox 모듈로 flex container(부모)를 flex item(자식)으로 나누어 flexbox layout을 구성한다.
