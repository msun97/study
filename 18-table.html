HTML table

정의

table 요소로 HTML 문서의 표(table)를 정의합니다.

Header 1 Header 2 Header 3
Row 1, Column 1 Row 1, Column 2 Row 1, Column 3
Row 2, Column 1 Row 2, Column 2 Row 2, Column 3
Row 3, Column 1 Row 3, Column 2 Row 3, Column 3

이때 td는 테이블 데이터, tr은 각 테이블 행, th는 테일 헤더입니다.

테이블 border

border 속성으로 table 요소의 border를 설정할 수 있습니다.

Header 1 Header 2 Header 3
Row 1, Column 1 Row 1, Column 2 Row 1, Column 3
Row 2, Column 1 Row 2, Column 2 Row 2, Column 3
Row 3, Column 1 Row 3, Column 2 Row 3, Column 3

border-collapse로 이중 테두리를 피할 수 있습니다.

Header 1 Header 2 Header 3
Row 1, Column 1 Row 1, Column 2 Row 1, Column 3
Row 2, Column 1 Row 2, Column 2 Row 2, Column 3
Row 3, Column 1 Row 3, Column 2 Row 3, Column 3

스타일 적용도 가능합니다.

Header 1 Header 2 Header 3
Row 1, Column 1 Row 1, Column 2 Row 1, Column 3

테이블 테두리 둥글게 하려면 border-radius를 사용합니다.

Header 1 Header 2 Header 3
Row 1, Column 1 Row 1, Column 2 Row 1, Column 3

border-style로 점선의 모양을 설정할 수 있습니다.

Header 1 Header 2 Header 3
Row 1, Column 1 Row 1, Column 2 Row 1, Column 3

border-color로 테두리의 색을 설정할 수 있습니다.

Header 1 Header 2 Header 3
Row 1, Column 1 Row 1, Column 2 Row 1, Column 3

테이블 크기

width, height 속성으로 table 요소의 크기를 조절할 수 있습니다.

Row 1, Column 1 Row 1, Column 2 Row 1, Column 3

padding and gap

padding, margin, border-spacing, border-collapse, gap 속성으로 table 요소의 padding, margin, border-spacing, border-collapse, gap을 조절할 수 있습니다.

Header 1 Header 2 Header 3

셀 padding

셀 간격

Colspan & Rowspan

colspan, rowspan 속성으로 table cell을 여러 행 및 열에 걸쳐 설정할 수 있다.

Row 1, Column 1 Row 1,2, Column 2 Row 1, Column 2
Row 2, Column 1 Row 2, Column 3

col-group tag를 사용하여 table header cell(th)를 그룹화할 수 있습니다.
